# основные паттерны регулярных выражений (классы символов)

- character classes - это специальные обозначения, которые позволяют искать определённые типы символов, а не конкретную строку

## 1. `\d` - цифры

```js
const patternDigits = /\d/;
```

- любая цифра от 0 до 9

### пример

```js
const textDigits = 'abc123';

console.log(patternDigits.test(textDigits)); // true
```

## 2. `\D` - не цифры

```js
const patternNotDigits = /\D/;
```

- любой символ, кроме цифры

## 3. `\w` - слово

```js
const patternWord = /\w/;
```

- означает:
  - буквы (a-z, A-Z)
  - цифры (0-9)
  - `_`

### пример

```js
const textWord = 'hello_123';

console.log(patternWord.test(textWord)); // true
```

## 4. `\W` - не слово

```js
const patternNotWord = /\W/;
```

- означает всё, кроме a-z, A-Z, 0-9, `_`

### пример

```js
console.log(/\W/.test('hello!')); // true (есть `!`)
```

## 5. `\s` - пробельные символы

```js
const patternSpace = /\s/;
```

- включает:
  - пробел `" "`
  - таб `\t`
  - перенос строки `\n`

### пример

```js
console.log(/\s/.test('hello world')); // true
```

## 6. `\S` — не пробельные символы

```js
const patternNotSpace = /\S/;
```

- любой символ, кроме пробельного

## 7. `.` — любой символ

```js
const patternAny = /./;
```

- означает: любой символ (кроме переноса строки)

### пример

```js
console.log(/./.test('a')); // true
console.log(/./.test('1')); // true
```

## 8. `[]` - собственный character class

- его можно задать самому

```js
const patternCustom = /[abc]/;
```

- означает: найти один символ: `a` или `b` или `c`

### пример

```js
console.log(/[abc]/.test('dog')); // false
console.log(/[abc]/.test('cat')); // true
```

### диапазоны

```js
const patternRange = /[a-z]/;
```

- означает: любая маленькая буква

### частые диапазоны:

- `[a-z]` — маленькие буквы

- `[A-Z]` — большие буквы

- `[0-9]` — цифры

### комбинации

```js
const patternCombo = /[a-zA-Z0-9]/;
```

- означает: любая буква или цифра

### отрицание внутри []

```js
const patternExclude = /[^0-9]/;
```

- означает: любой символ, кроме цифры

### пример

```js
console.log(/[^0-9]/.test('123')); // false
console.log(/[^0-9]/.test('abc')); // true
```

### частая ошибка

`/[abc]/`

- это не строка "abc"
- это: один символ: a или b или c
