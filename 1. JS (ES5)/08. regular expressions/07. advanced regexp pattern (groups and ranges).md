# продвинутые паттерны регулярных выражений (группы и диапазоны)

## 1. группы (groups) `()`

- создаются с помощью скобок `(...)`

- нужны для того, чтобы:
  1. сгруппировать часть выражения
  2. получить кусок строки
  3. переиспользовать

### пример

```js
const patternGroup = /(ab)+/;

console.log('abab'.match(patternGroup)); // "ab" повторяется 1 или более раз
```

### захватывающие группы

```js
const patternCapture = /(a)(b)/;

const resultCapture = 'ab'.match(patternCapture);

console.log(resultCapture);
```

- результат:

```
[
  "ab", // полное совпадение
  "a",  // 1-я группа
  "b"   // 2-я группа
]
```

### группы + квантификаторы

```js
const patternRepeatGroup = /(ab)+/; // повторяем именно "ab", а не `a` и `b` отдельно
```

```js
/ab+/   // a + много b
/(ab)+/ // повтор "ab"
```

### незахватывающие группы

```js
const patternNonCapture = /(?:ab)+/; // `?:` означает: не сохранять в результат
```

- когда использовать
  - если группа нужна только для логики
  - чтобы не засорять результат

## 2. диапазоны (ranges) `[]`

### комбинированные диапазоны

```js
const patternComboRange = /[a-zA-Z0-9]/; // буквы + цифры
```

### диапазоны с исключением

```js
const patternExcludeRange = /[^a-z]/; // всё кроме маленьких букв
```

### диапазоны + квантификаторы

```js
const patternWordLength = /[a-z]{3,5}/; // слово из 3-5 букв

console.log('hello'.match(/[a-z]{3,5}/)); // "hello"
```

## практический пример

- проверка простого username

```js
const patternUsername = /^[a-zA-Z0-9_]{3,10}$/;
```

- `^` → начало
- `[a-zA-Z0-9_]` → допустимые символы
- `{3,10}` → длина
- `$` → конец

=> username от 3 до 10 символов, только буквы/цифры/`_`
