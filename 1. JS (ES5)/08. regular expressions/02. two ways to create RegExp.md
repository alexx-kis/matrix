# 2 способа создать регулярное выражение

- в JS можно создать регулярное выражение двумя способами:
  1. литерал
  2. конструктор

## 1. литерал

- это самый простой и быстрый способ

```js
const patternLiteral = /\d+/;
```

- `/` - начало и конец регулярного выражения
- `\d+` - шаблон (одна или более цифр)

```js
const text = 'I have 42 apples';

console.log(/\d+/.test(text)); // true
```

### особенности литерала

- компактно, удобно для статических шаблонов
- можно сразу добавлять флаги после закрывающего `/`, например:
  - `/abc/i` - `i` === case-insensitive (игнорировать регистр)
  - `/abc/g` - `g` === global (найти все совпадения)
  - `/abc/m` - `m` === многострочный режим

## 2. конструктор

- можно создавать через функцию-конструктора

```js
const patternConstructor = new RegExp('\\d+');
```

- в строке нужно экранировать обратный слэш (`\`) двойным слэшем (`\\`)
- то есть `\d` -> `"\\d"`

### пример использования

```js
const text = 'I have 42 apples';

const pattern = new RegExp('\\d+');
console.log(pattern.test(text)); // true
```

### когда использовать конструктор

- когда регулярное выражение создаётся динамически, из переменной:

```js
const userInput = 'abc';
const dynamicPattern = new RegExp(userInput, 'i');

console.log(dynamicPattern.test('ABC')); // true
```

- если использовать литерал, динамический ввод не получится:

```js
const userInput = 'abc';
const wrongPattern = /userInput/i; // ищет буквально "userInput"
```

## сравнение способов

```
Способ	        Пример	                    Применение
Литерал	        /abc/i	                    Статические, заранее известные шаблоны
Конструктор	    new RegExp("abc", "i")	    Динамические, строятся из переменной
```
