# IF condition

- инструкция if выполняет код, если указанное условие истинно
- если условие не истинно, то может быть выполнена другая инструкция

```js
if (condition) {
  // ...
} else {
  // ...
}
```

- можно использовать несколько условий (else if):

```js
const age = 18;

if (age < 18) {
  console.log('Мало лет');
} else if (age === 18) {
  console.log('Идеальный возраст');
} else {
  console.log('Взрослый');
}
```

## короткая запись

- если в блоке только одно выражение, фигурные скобки можно опустить

```js
if (condition) console.log('выполнится');
else console.log('не выполнится');
```

## тернарный оператор

- это коротка форма if-else
- можно вкладывать друг в друга, но лучше не надо
- часто используется для присваивания в переменную какого-то значения без использования let и конструкции if-else

```js
const age = 18;
const message = age >= 18 ? 'Взрослый' : 'Малолетка';
console.log(message);
```

## Truthy / Falsy значения

- "истинные" значения - все, кроме: false, 0, null, undefined, NaN, ''

## разница между == и ===

- == проверяет равенство с приведением типов (если оно возможно)
- например 2 == '2' // true

- === проверяет равенство с учётом типа
- например 2 === '2' //false
