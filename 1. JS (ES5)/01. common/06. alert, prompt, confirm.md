# alert, prompt, confirm

- это встроенные методы браузера (в nodeJS их нет)

## 1. alert()

- показывает модальное окно с сообщением и кнопкой ОК
- ничего не возвращает
- блокирует выполнение кода, пока пользователь не нажал ОК

```js
alert('hello');
```

### важные детали

- это синхронная функция
- пока окно открыто, JS останавливается
- редко используется на продакшене (обычно для отладки или учебных задач)

## 2. prompt()

- показывает окно с сообщением, полем ввода и кнопками OK и Cancel
- возвращает строку (!), если нажат OK, и null, если нажат Cancel

- в связи с этим проблемой могут стать операции с числами - введённые числа будут преобразованы в строки, и все операции с ними будут производиться, как со строками

```js
let a = prompt('Введите число');
let b = prompt('Введите число');

console.log(a + b);
```

- если ввести 2 и 3, то результат будет '23'
- поэтому нужно приводить тип к числу

```js
let a = Number(prompt('Введите число'));
let b = Number(prompt('Введите число'));

console.log(a + b); // 5
```

### особенность

```js
let name = prompt('Введите имя', 'Гость');
```
- в поле ввода уже будет 'Гость'

## 3. confirm()

- показывает окно с сообщением и кнопками OK и Cancel
- возвращает true, если нажат OK, и false, если нажат Cancel

```js
let result = confirm('Вы уверены?');
```

- операторы сравнения приводят строки к числу
- если строка состоит из букв, она преобразуется в NaN
- любое сравнение с NaN даёт false
