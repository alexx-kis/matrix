# writeable properties

## рассмотренные ранее свойства

```js
element.clientWidth;
element.clientHeight;

element.offsetWidth;
element.offsetHeight;

element.scrollWidth;
element.Height;
```

- являются read-only

## изменяемые свойства

### scrollTop и scrollLeft

- показывает текущую прокрутку элемента по вертикали и горизонтали, соответственно

```js
// можно читать
console.log(chatContainer.scrollTop); // 100

// можно изменять
chatContainer.scrollTop = 300;

// после этого элемент прокрутится вниз на 300px

// прокрутка в начало
chatContainer.scrollTop = 0;

// прокрутка в конец
chatContainer.scrollTop = chatContainer.scrollHeight;
```
