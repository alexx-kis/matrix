# what is DOM

- DOM (Document Object Model) – это объектное представление HTML-документа, которое браузер создает после загрузки страницы

- проще говоря:
  - HTML – это текстовый файл
  - браузер читает этот текст
  - на его основе строит структуру объектов в памяти
  - эта структура называется DOM

- например, есть HTML:

```html
<!DOCTYPE html>
<html>
  <body>
    <h1>Hello</h1>
    <button>Click me</button>
  </body>
</html>
```

- после разбора браузер создает примерно такую структуру:

```
Document
│
└── html
    │
    └── body
        │
        ├── h1
        │   └── "Hello"
        │
        └── button
            └── "Click me"
```

- каждый элемент становится объектом, с которым можно работать через JS

## зачем нужен DOM?

- без DOM JS не смог бы взаимодействовать со страницей

- например:

```js
const button = document.querySelector('button');
button.textContent = 'New text';
```

- JS находит объект кнопки в DOM и изменяет его содержимое

## DOM – это дерево

- DOM часто называют DOM Tree

- для такого HTML:

```html
<body>
  <div>
    <p>Hello</p>
    <span>World</span>
  </div>
</body>
```

- дерево выглядит так:

```
body
│
└── div
    │
    ├── p
    │   └── "Hello"
    │
    └── span
        └── "World"
```

- поэтому часто говорят:
  - parent
  - child
  - sibling

```html
<div>
  <p>Hello</p>
  <span>World</span>
</div>
```

- здесь:
  - `div` – родитель для `p` и `span`
  - `p` и `span` – дети `div`
  - `p` и `span` – siblings друг для друга

## главный объект DOM

- точка входа в DOM: document – это объект, представляющий весь HTML-документ
- например,

```js
console.log(document);
console.log(document.body);
console.log(document.documentElement);
```

```
document
 └── html (documentElement)
      └── body
```

### HTML и DOM – это не одно и то же

- HTML:

```html
<button>Click</button>
```

- просто текст

- DOM

```
HTMLButtonElement;
```

- объект в памяти браузера

- когда пишешь:

```js
const button = document.querySelector('button');
```

- получаешь не строку из HTML, а объект HTMLButtonElement
- у него есть свойства и методы:

```js
button.textContent;
button.classList;
button.style;
button.addEventListener;
```

## DOM можно изменять

```js
// можно создавать элементы:
const div = document.createElement('div');

// добавлять их:
document.body.append(div);

// удалять:
div.remove();

// изменять текст:
div.textContent = 'Hello';

// менять классы:
div.classList.add('active');
```

- то есть JS постоянно работает именно с DOM, а не напрямую с HTML-файлом

## как происходит загрузка страницы

```
HTML
  ↓
Парсинг браузером
  ↓
Создание DOM
  ↓
JavaScript получает доступ к DOM
  ↓
Изменения страницы
```

## что нужно запомнить

- DOM – это дерево объектов, которое браузер строит из HTML-документа

- HTML -> преобразуется в DOM
- DOM существует в памяти браузера
- каждый HTML-элемент становится объектом
- JS взаимодействует со страницей через DOM
- DOM имеет древовидную структуру
- главная точка входа – объект `document`
