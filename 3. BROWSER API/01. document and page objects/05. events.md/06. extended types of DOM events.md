# extended types of DOM events

## Pointer events

- это современная система событий для работы с указателем
- раньше были отдельно Mouse Events и Touch Events

- проблема в том, что для каждой группы был свой набор событий - приходилось писать разный код

- Pointer Events объединили все в одну систему, теперь один обработчик работает для мыши, пальца, стилуса, трекпада

- pointerdown ⇆ mousedown
- pointerup ⇆ mouseup
- pointerenter ⇆ mouseenter
- pointerleave ⇆ mouseleave

## Wheel events

- события колеса мыши

### wheel

- срабатывает при прокрутке колеса

```js
galleryElement.addEventListener('wheel', (event) => {
  console.log(event.deltaY);
});
```

- можно получить:

```js
event.deltaX;
event.deltaY;
event.deltaZ;
```

- частое применение:
  - кастомный скролл
  - масштабирование
  - горизонтальная прокрутка

## Clipboard Events

- работа с буфером обмена

### copy

```js

});
```

### cut

### paste

- можно получить вставленные данные

```js
document.addEventListener('paste', (event) => {

```

## Drag and Drop Events

- встроенная система перетаскивания
- используется в:
  - канбан досках
  - загрузке файлов
  - перетаскивании карточек

### dragstart - начало перетаскивания

```js
taskCardElement.addEventListener('dragstart', () => {
  console.log('start');
});
```

### drag - во время перетаскивания

### dragenter - курсор с объектом вошел в зону

### dragleave - курсор покинул зону

### dragover - происходит над зоной сброса

### drop - объект сброшен

- чтобы работал drop, обычно нужно:

```js
event.preventDefault(); // в обработчике dragover
```

### dragend - окончание перетаскивания

## Touch Events

- события мобильных устройств

### touchstart - палец коснулся экрана

```js
mobilePanelElement.addEventListener('touchstart', () => {
  console.log('touch start');
});
```

### touchmove - палец двигается

### touchend - палец убран

### touchcancel - система прервала касание

- например, входящий звонок или системный жест

## Composition Events

- очень редко используются
- особенно если речь про международное приложение

- когда пользователь вводит текст через китайский/японский/корейский/IME-клавиатуры происходит специальный процесс композиции символов

### compositionstart - начало композиции

### compositionupdate - обновление композиции

### compositionend - завершение композиции

```js
messageInputElement.addEventListener('compositionend', () => {
  console.log('finished');
});
```

## Selection Events

- работа с выделением текста

### select

- пользователь выделил текст внутри поля

```js
commentInputElement.addEventListener('select', () => {
  console.log('selected');
});
```

- полезно для редакторов, IDE, форматирования текста

## Context Menu Events

### contextmenu

- правый клик мышью

```js
document.addEventListener('contextmenu', (event) => {
  event.preventDefault();

  console.log('custom menu');
});
```

- часто используется для кастомных контекстных меню, онлайн-редакторов, графических приложений

### animationiteration - новый цикл анимации

## Media Events

- события аудио и видео

```html
<video id="promoVideo"></video>
```

### play - видео начало воспроизводиться

### pause - видео остановлено

### ended - видео завершилось

### volumechange - изменилась громкость

```js
promoVideoElement.addEventListener('ended', () => {
  console.log('video finished');
});
```
