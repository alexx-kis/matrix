# basic types of DOM events

- DOM-событий очень много, но на собесах обычно ожидают понимание основных групп событий и самых популярных представителей каждой группы

## основные группы DOM-событий

1. Mouse events
2. PointerEvents
3. Keyboard events
4. Focus events
5. Form events
6. Input events
7. Clipboard events
8. Drag and Drop events
9. Touch events
10. Window / Document events

## 1. Mouse events - события мыши

- самая распространенная группа

### click - нажал ЛКМ, отпустил ЛКМ

```js
const loginButtonElement = document.querySelector('#loginButton');

loginButtonElement.addEventListener('click', () => {
  console.log('clicked');
});
```

### dblclick - двойной клик

### mousedown - кнопка мыши нажата

### mouseup - кнопка мыши отпущена

### mousemove - движение мыши

```js
document.addEventListener('mousemove', () => {
  console.log('moving');
});
```

- может вызываться сотни раз в секунду

### mouseover - курсор вошел в элемент

```js
cardElement.addEventListener('mouseover', () => {
  console.log('hover');
});
```

### mouseout - курсор покинул элемент

## 2. pointer events

- современная замена многим mouse и touch событиям
- позволяет работать одинаково с:
  - мышью
  - стилусом
  - пальцем на тачскрине

### pointerdown - аналог mousedown

### pointerup - аналог mouseup

### pointermove - аналог mousemove

### pointerenter - курсор вошел в элемент

### pointerleave - курсор вышел из элемента

- на современных проектах часто используют именно Pointer Events

## 3. Keyboard events

### keydown - клавиша нажата

```js
document.addEventListener('keydown', (event) => {
  console.log(event.key);
});
```

### keyup - клавиша отпущена

```js
document.addEventListener('keyup', (event) => {
  console.log(event.key);
});
```

### keypress - устаревшее событие

## 4. Focus events

- связаны с получением и потерей фокуса

### focus - элемент получил фокус

```html
<input />
```

- пользователь кликнул в поле -> focus

### blur - элемент потерял фокус

```js
emailInputElement.addEventListener('focus', () => {
  console.log('focused');
});

emailInputElement.addEventListener('blur', () => {
  console.log('unfocused');
});
```

## 5. Form events

### submit - отправка формы

```js
registrationFormElement.addEventListener('submit', () => {
  console.log('submit');
});
```

### reset - сброс формы

```js
registrationFormElement.addEventListener('reset', () => {
  console.log('reset');
});
```

## 6. Input events

### input - значение изменилось

```js
searchInputElement.addEventListener('input', (event) => {
  console.log(event.target.value);
});
```

- срабатывает при каждом изменении, например:

```
a
ab
abc
abcd
```

### change - значение изменилось и изменение подтверждено пользователем

- например, пользователь ввел текст в input
- пока курсор внутри поля, change не вызывается; когда потерял фокус, change вызывается

## 7. Clipboard event

- работа с буфером обмена

### copy - копирование

```js
document.addEventListener('copy', () => {
  console.log('copy');
});
```

### cut - вырезание

```js
document.addEventListener('cut', () => {
  console.log('cut');
});
```

### paste - вставка

```js
document.addEventListener('paste', () => {
  console.log('paste');
});
```

## 8. Drag and Drop events

- перетаскивание элемента

- основные события:
  - dragstart
  - drag
  - dragenter
  - dragover
  - drop
  - dragend

- примеры использования:
  - загрузка файлов через drag-and-drop
  - канбан-доски
  - перетаскивание карточек

## 9. Touch events

- для мобильных устройств

- основные:
  - touchstart
  - touchmove
  - touchend

- сейчас часто заменяются на Pointer Events

## 10. Window / Document events

- события документа и окна браузера

### DOMContentLoaded - DOM построен

```js
document.addEventListener('DOMContentLoaded', () => {
  console.log('DOM ready');
});
```

### load - загружено всё

- HTML
- CSS
- изображения
- шрифты

### resize - размер окна изменился

```js
window.addEventListener('resize', () => {
  console.log(window.innerWidth);
});
```

### scroll - прокрутка страницы

```js
window.addEventListener('scroll', () => {
  console.log(window.scrollY);
});
```
