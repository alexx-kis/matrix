# form events

## основные form events

### submit

- главное событие формы, срабатывает при отправке формы

```html
<form id="loginForm">
  <input type="text" />

  <button type="submit">Login</button>
</form>
```

```js
const loginFormElement = document.querySelector('#loginForm');

loginFormElement.addEventListener('submit', () => {
  console.log('submitted');
});
```

#### когда возникает submit

- при нажатии на кнопку внутри формы (type="submit")
- при нажатии Enter внутри поля формы

- после submit страница перезагружается, потому что это стандартное поведение браузера
- чтобы страница не перезагружалась, нужно вызывать `event.preventDefault()`

```js
loginFormElement.addEventListener('submit', (event) => {
  event.preventDefault();

  console.log('submitted');
});
```

- для обработки события submit лучше подписываться на форму, а не на ее кнопку
- потому что submit может произойти:
  - через кнопку
  - через Enter
  - через JS
- а событие submit всегда будет поймано формой

### reset

- срабатывает при сбросе формы

```html
<form id="settingsForm">
  <input />

  <button type="reset">Reset</button>
</form>
```

```js
settingsFormElement.addEventListener('reset', () => {
  console.log('reset');
});
```

- после reset значения возвращаются к первоначальным

### input

- срабатывает при каждом изменении значения

```html
<input id="searchInput" />
```

```js
const searchInputElement = document.querySelector('#searchInput');

searchInputElement.addEventListener('input', (event) => {
  console.log(event.target.value);
});
```

- где используется input:
  - живой поиск
  - валидация
  - автодополнение
  - маски ввода

### change

```html
<input id="emailInput" />
```

```js
emailInputElement.addEventListener('change', handleEmailChange);
```

- событие вызовется, когда пользователь покинет поле -> обработчик выполнится

### focus

- вызывается, когда элемент получил фокус

```js
usernameInputElement.addEventListener('focus', () => {
  console.log('focus');
});
```

### blur

- когда элемент потерял фокус

```js
usernameInputElement.addEventListener('blur', () => {
  console.log('blur');
});
```

- часто используется для валидации поля после завершения ввода

#### важная особенность

- события focus и blur не всплывают, то есть делегирование на них не работает
- есть специальные аналоги `focusin` и `focusout`, которые всплывают
