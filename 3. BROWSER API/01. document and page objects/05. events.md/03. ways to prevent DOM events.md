# ways to prevent DOM events

## стандартное поведение браузера

- у многих HTML-элементов уже есть встроенное поведение
- например, при клике по ссылке браузер по ней переходит
- при нажатии на кнопку внутри формы браузер отправляет форму и перезагружает страницу
- при клике на чекбокс браузер меняет его состояние

## preventDefault()

- позволяет отменить стандартное действие браузера
- синтаксис:

```js
event.preventDefault();
```

## как понять, можно ли отменить событие

- у объекта события есть свойство `event.cancelable`

```js
document.addEventListener('click', (event) => {
  console.log(event.cancelable); // если true, то событие можно отменить через `preventDefault()`
});
```
