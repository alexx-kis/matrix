# ways to get DOM elements

- когда JS работает со страницей, ему сначала нужно найти элемент в DOM, а уже потом что-то с ним делать
- для поиска элементов существует несколько методов

1. getElementById() - возвращает HTMLElement / null

2. getElementByClassName() - возвращает HTMLCollection

3. getElementByTagName() - возвращает HTMLCollection

4. querySelector() - возвращает первый найденный элемент, можно искать по тегу, классу (.), id (#), атрибутам ([]), вложенностям

5. querySelectorAll() - возвращает NodeList всех найденных элементов, который можно перебирать циклом
