# cookie removing

- в браузере нет механизма типа:

```js
document.cookie.remove('cookie');
```

- удаление cookie происходит через установку ей срока жизни в прошлом
- то есть браузеру говорят: "эта cookie уже просрочена, удали ее"

## Expires

- допустим, существует cookie:

```js
document.cookie = 'theme=dark';
```

- для удаления:

```js
document.cookie = 'theme=dark; Expires=Thu, 01 Jan 1970 00:00:00 GMT';
```

- эта дата традиционно используется, как дата в прошлом, браузер сразу удалит cookie

## Max-Age

- более современный способ:

```js
document.cookie = 'theme=dark; Max-Age=0';
```

- срок жизни закончится прямо сейчас, браузер удалит cookie

## сервер удаляет cookie так же

```http
Set-Cookie:
sessionId=abc123;
Max-Age=0

// или

Set-Cookie:
sessionId=abc123;
Expires=Thu, 01 Jan 1970 00:00:00 GMT
```

- удаление - это фактически новая установка cookie

## очень важный нюанс: Path

- может возникнуть ситуация:

```js
document.cookie = 'theme=dark; Path=/';

// затем удаление:
document.cookie = 'theme=dark; Max-Age=0';
```

- но cookie не исчезает, потому что она идентифицируется не только по имени
- у нее есть набор параметров:
  - name
  - domain
  - path

- для "удаления" нужно указать те же значения

- например:

```js
// создание:
document.cookie = 'theme=dark; Path=/';

// удаление:
document.cookie = 'theme=dark; Path=/; Max-Age=0';
```

- если path не совпадет, то браузер будет считать это другой cookie

- то же касается Domain

## как выглядит logout

- допустим, авторизация хранится в cookie

```http
sessionId=abc123
```

- при logout сервер отвечает:

```http
Set-Cookie: sessionId=;
Max-Age=0;
HttpOnly;
Secure
```

- после этого браузер удаляет cookie

## удаление HttpOnly

- если cookie была создана так:

```http
Set-Cookie: sessionId=abc123;
HttpOnly
```

- то JS ее не видит
- поэтому `document.cookie = 'sessionId=; Max-Age=0';` не поможет
- удалять такую cookie должен сервер:

```http
Set-Cookie: sessionId=;
Max-Age=0;
HttpOnly
```

## как удаляют cookie библиотеки

- например, библиотека `js-cookie`

```js
Cookies.remove('theme');
```

- на самом деле внутри она делает примерно следующее:

```js
document.cookie = 'theme=; Max-Age=0';
```

## проверка удаления

- в DevTools: Application -> Cookies

- можно увидеть список cookie
- после удаления запись исчезнет
