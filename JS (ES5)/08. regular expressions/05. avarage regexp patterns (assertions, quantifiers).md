# average regexp pattern (assertions, quantifiers)

## 1.quantifiers (квантификаторы)

- сколько раз должен повторяться символ

### `+` - один или больше

```js
const patternPlus = /\d+/; // одна или больше цифр

const textPlus = 'abc123xyz';

console.log(textPlus.match(/\d+/)); // ['123']
```

### `*` - ноль или больше

```js
const patternStar = /\d*/; // может быть 0, 1 или много цифр
```

#### важно

```js
console.log('abc'.match(/\d*/)); // [""] пустая строка, потому что "0 цифр" — тоже подходит
```

### `?` - ноль или один

```js
const patternOptional = /colou?r/; // "u" может быть, а может не быть

console.log('color'.match(/colou?r/)); // match
console.log('colour'.match(/colou?r/)); // match
```

### `{n}` — ровно n раз

```js
const patternExact = /\d{3}/;

console.log('12345'.match(/\d{3}/)); // "123"
```

### `{n,m}` — от n до m

```js
const patternRange = /\d{2,4}/; // от 2 до 4 цифр
```

### жадность (greedy)

- по умолчанию квантификаторы - жадные:

```js
const textGreedy = '12345';

console.log(textGreedy.match(/\d{2,4}/)); // "1234"
```

- берёт максимум (4), а не минимум (2)

## 2. assertions (утверждения)

- они не захватывают символы
- они проверяют позицию

### `^` - начало строки

```js
const patternStart = /^Hello/; // строка должна начинаться с `"Hello"`

console.log(/^Hello/.test('Hello world')); // true
console.log(/^Hello/.test('world Hello')); // false
```

### `$` - конец строки

```js
const patternEnd = /world$/; // строка должна заканчиваться на `"world"`

console.log(/world$/.test('Hello world')); // true
console.log(/world$/.test('world Hello')); // false
```

#### комбинация

```js
const patternExactWord = /^hello$/; // строка должна быть ровно "hello"
```

### word boundary `/b`

```js
const patternWordBoundary = /\bcat\b/; // слово "cat" целиком

console.log(/\bcat\b/.test('cat')); // true
console.log(/\bcat\b/.test('catalog')); // false
```
