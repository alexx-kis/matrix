# 2 метода RegExp

## 0. что такое regular expression (регулярное выражение)

- это специальная строка-шаблон, которая используется для:
  - поиска текста
  - проверки (валидации) строки
  - замены части строки
  - извлечения данных

### пример

- допустим, есть строка

```js
const textValue = 'Hello 123 world';
```

- и нужно найти числа внутри строки
- с помощью RegExp можно описать шаблон:

```js
/\d+/;
```

- `\d` - любая цифра
- `+` - одна или больше

- то есть: "найди подряд идущие цифры

### где используется RegExp$

- проверка email
- проверка пароля
- поиск слов в тексте
- парсинг данных
- замена текста

## 1. 2 метода RegExp

- в JS есть 2 основных метода у объекта RegExp, которые используются чаще всего

### 1. `test()`

- проверяет, если совпадение

- возвращает true - если есть, false - если нет

```js
const regExpCheck = /\d+/;

const resultCheck = regExpCheck.test('Hello 123');

console.log(resultCheck);
```

#### что происходит:

- ищем цифры в строке
- "123" найдено
- результат: true

```js
const regExpLetters = /abc/;

console.log(regExpLetters.test('xyz')); // false
```

#### когда использовать `.test()`

1. просто проверить условие
2. валидация (email, пароль итд)

## 2. `.exec()`

- выполняет поиск и возвращает подробную информацию

- возвращает:
  - `null` - если ничего нет
  - массив - если найдено

```js
const regExpExec = /\d+/;

const resultExec = regExpExec.exec('Hello 123 world');

console.log(resultExec);
```

- результат:

```
[
  "123",        // найденное совпадение
  index: 6,     // позиция в строке
  input: "Hello 123 world" // исходная строка
]
```

## 3. важное различие

```
Метод	      Что делает	          Возвращает

.test()	    проверяет наличие	    true / false
.exec()	    ищет и даёт данные	  массив / null
```
