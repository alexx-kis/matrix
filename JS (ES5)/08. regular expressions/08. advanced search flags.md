# продвинутые флаги поиска

## 1. `m` (multiline)

- меняет поведение:
  - `^` (начало)
  - `$` (конец)

### без `m`

```js
const textMulti = 'Hello\nWorld';

console.log(/^World/.test(textMulti)); // false
```

- потому что `^` смотрит только на начало всей строки

### с `m`

```js
const patternMultiline = /^World/m;

console.log(patternMultiline.test(textMulti)); // true
```

- каждая строка считается отдельно
- `^` и `$` работают для каждой строки

## 2. `s` (dotAll)

- меняет поведение `.`

### без `s`

```js
const textDot = 'Hello\nWorld';

console.log(/Hello.World/.test(textDot)); // false
```

- потому что `.` НЕ матчится с `\n`

### с `s`

```js
const patternDotAll = /Hello.World/s;

console.log(patternDotAll.test(textDot)); // true
```

- теперь `.` = вообще любой символ (включая перенос строки)

## 3. `u` (unicode)

- включает полноценную поддержку Unicode

### без `u`

```js
console.log(/😊/.test('😊'));
```

- может работать нестабильно в сложных случаях

### с `u`

```js
console.log(/😊/u.test('😊'));
```

- корректная работа с:
  - emoji
  - редкими символами
  - суррогатными парами

### где реально важно

```js
const patternUnicode = /^\p{L}+$/u; // только буквы любого языка
```

## 4. `y` — sticky

- похож на `g`, но ищет строго с позиции `lastIndex`

### Пример

```js
const patternSticky = /test/y;

patternSticky.lastIndex = 5;

console.log(patternSticky.exec('hello test'));
```

- сработает только если совпадение ровно с позиции 5
