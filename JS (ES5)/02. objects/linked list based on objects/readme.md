# linked list based on objects

## 1️⃣ что такое linked list

- linked list (связанный список) - это структура данных, где элементы (узлы, nodes) хранятся не подряд в памяти, а каждый узел хранит значение и ссылку на следующий узел
- противоположность - массив: в массиве все элементы идут подряд, и доступ к элементу по индексу быстрый
- в связанном списке доступ к элементу по индексу требует прохода по списку от головы (head) до нужного узла

```
[1] -> [2] -> [3] -> null
```

- `[1]` - первый узел (head), хранит значение `1` и ссылку на следующий `[2]`
- последний узел ссылается на `null` - конец списка

## 2️⃣ node (узел) на объектах

- в JS можно представить узел так:

```js
const node = {
  value: 10,
  next: null,
};
```

- `value` - данные узла
- `next` - ссылка на следующий узел (или `null`, если конец списка)

✅ `next` - это обычное свойство объекта, которое хранит ссылку на следующий узел

```js
const nodeA = {
  value: 1,
  next: null,
};

const nodeB = {
  value: 2,
  next: nodeA,
};
```

- получается цепочка `nodeB` -> `nodeA` -> `null`
