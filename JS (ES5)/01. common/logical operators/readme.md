# логические операторы

## 1. логическое ИЛИ (||)

- возвращает первое истинное значение или последнее, если все ложные
- если в выражении хотя бы один операнд истинный, то true

- работает как операция сложения, где истинный операнд = 1, ложный = 0

1 + 0 = 1 // true 1 + 1 = 1 // true 0 + 0 = 0 // false

```js
console.log(0 || 'hello'); // 'hello'
console.log(null || undefined); // undefined
console.log('hi' || 0); // 'hi'
```

## 2. логическое И (&&)

- возвращает первое ложное значение или последнее, если все истинные
- если в выражении оба (все) операнды истинные, то true

- работает как операция умножения, где истинный операнд = 1, ложный = 0

0 _ 0 = 0 // false 1 _ 0 = 0 // false 1 \* 1 = 1 // true

```js
console.log(5 && 'hello'); // 'hello'
console.log(0 && 'hello'); // 0
console.log(true && 10); // 10
```

## 3. short-circuit (короткое замыкание)

- js не вычисляет правую часть, если уже понятно, какой результат.

- Пример с ||

```js
true || console.log('не выполнится');
```

Правая часть не выполнится.

- Пример с &&

```js
false && console.log('не выполнится');
```

Правая часть не выполнится.

## 4. использование

### дефолтное значение через ||

```js
let name;
console.log(name || 'Гость'); // "Гость"
```

### условное выполнение через &&

```js
isAdmin && showAdminPanel();
```

- работает, как короткий if

## 5. логическое НЕ (!)

- используется для инверсии значения (с учётом приведения типов)

```js
!true; // false
!0; // true
!''; // true
```

- двойное отрицание используется для преобразования в boolean

```js
!!0; // false
!!'hello'; // true
```

## 6. приоритет операторов

! > && > ||
