# examples of async functions

- async function - это функция, внутри которой можно использовать await для ожидания Promise

```js
async function loadData() {
  //
}
```

- async function всегда возвращает Promise
- даже если вернуть обычное значение

```js
async function getNumberValue() {
  return 10;
}

// фактически это превращается в:

function getNumberValue() {
  return Promise.resolve(10);
}

const resultPromise = getNumberValue();

console.log(resultPromise); // Promise { 10 }
```

### await

- говорит: "подожди завершения Promise"

### пример:

```js
function waitOneSecond() {
  return new Promise((resolveFunction) => {
    setTimeout(() => {
      resolveFunction('done');
    }, 1000);
  });
}

async function runExample() {
  const resultValue = await waitOneSecond();

  console.log(resultValue);
}

runExample();
```

- вывод через секунду: done

- с `await` мы получаем готовый результат Promise
- `await` работает только внутри async function

### несколько await

```js
function getUserName() {
  return Promise.resolve('Alex');
}

function getUserAge() {
  return Promise.resolve(25);
}

async function loadProfileData() {
  const userName = await getUserName();

  const userAge = await getUserAge();

  console.log(userName);
  console.log(userAge);
}

loadProfileData();
```

- async/await делает код похожим на синхронный - это главная причина популярности

## обработка ошибок

- используется обычный try-catch

```js
function loadSettingsData() {
  return Promise.reject('settings failed');
}

async function runSettingsLoader() {
  try {
    const settingsData = await loadSettingsData();

    console.log(settingsData);
  } catch (errorValue) {
    console.log(errorValue);
  }
}

runSettingsLoader();
```

### что происходит

1. await дождался Promise
2. Promise стал rejected
3. await выбросил ошибку
4. catch ее поймал

## await внутри цикла

```js
async function processItemsList() {
  await processItem('A');

  await processItem('B');

  await processItem('C');
}

processItemsList();
```

- все выведутся по порядку, так как каждый await останавливает async function
- await не блокирует весь JS, а останавливает только текущую функцию:

```js
function waitTask() {
  return new Promise((resolveFunction) => {
    setTimeout(resolveFunction, 2000);
  });
}

async function executeTask() {
  console.log('start');

  await waitTask();

  console.log('end');
}

executeTask();

console.log('outside');
```

- вывод:
```
start
outside
end
```

- потому что пока async function ждет Promise, остальной JS продолжает работать
