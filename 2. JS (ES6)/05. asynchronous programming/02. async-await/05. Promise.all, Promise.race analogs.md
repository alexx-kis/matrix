# Promise.all, Promise.race analogs

- async/await не имеет собственных аналогов для
  - Promise.all()
  - Promise.race()
  - Promise.allSettled()
  - Promise.any()

- даже при использовании async/await эти методы продолжают использоваться

## Promise.all

- запускает несколько Promise параллельно и ждет завершения всех

```js
// без Promise.all

async function loadData() {
  const profileData = await loadProfileData();
  const settingsData = await loadSettingsData();

  console.log(profileData);
  console.log(settingsData);
}

// что происходит:
// 1. ждем loadProfileData()
// 2. только потом запускаем loadSettingsData
// то есть выполнение последовательное

// с Promise.all

async function loadData() {
  const [profileData, settingsData] = await Promise.all([loadProfileData(), loadSettingsData()]);

  console.log(profileData);
  console.log(settingsData);
}

// что происходит:
// 1. оба promise запускаются сразу
// 2. ждем завершения обоих
// 3. получаем результаты
// обычно это быстрее
```

## самый частый паттерн

```js
const [userData, ordersData, notificationsData] = await Promise.all([loadUserData(), loadOrdersData(), loadNotificationsData()]);
```

- такой код встречается постоянно в React, NextJS, NodeJS

## Promise.race

- возвращает результат первого завершившегося promise

```js
//Promise

Promise.race([loadImageData(), loadFallbackData()]).then((resultData) => {
  console.log(resultData);
});

// async/await

async function loadData() {
  const resultData = await Promise.race([loadImageData(), loadFallbackData()]);

  console.log(resultData);
}
```

## пример timeout через Promise.race

```js
function createTimeoutPromise() {
  return new Promise((_, reject) => {
    setTimeout(() => {
      reject(new Error('Request timeout'));
    }, 5000);
  });
}

async function loadUserData() {
  const userData = await Promise.race([fetch('/api/user'), createTimeoutPromise()]);

  return userData;
}
```

- что произойдет:
  - если fetch успел раньше -> получим данные
  - если прошло 5с -> получим ошибку timeout

## Promise.allSettled

- ждет завершения всех Promise независимо от ошибок

```js
async function loadData() {
  const resultsArray = await Promise.allSettled([loadProfileData(), loadOrdersData(), loadSettingsData()]);

  console.log(resultsArray);
}
```

- результат выглядит так:

```js
[
  {
    status: 'fulfilled',
    value: 'profile',
  },
  {
    status: 'rejected',
    reason: 'error',
  },
  {
    status: 'fulfilled',
    value: 'settings',
  },
];
```

## Promise.any

- возвращает первый успешно завершившийся promise

```js
async function loadData() {
  const resultData = await Promise.any([loadMirrorServerA(), loadMirrorServerB(), loadMirrorServerC()]);

  console.log(resultData);
}
```
