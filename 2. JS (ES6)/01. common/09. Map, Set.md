# Map, Set

## 1. `Set`

- это коллекция уникальных значений
- то есть одинаковые значения не добавляются
- порядок сохраняется

### создание

```js
const uniqueNumbers = new Set();

// или сразу с данными:

const uniqueValues = new Set([1, 2, 3, 3, 4]); // {1, 2, 3, 4}
```

### основные методы

```js
// 1. добавить значение (`.add()`)

uniqueNumbers.add(10);
uniqueNumbers.add(10); // не добавится второй раз

// 2. проверить наличие (`.has()`)

uniqueNumbers.has(10); // true

// 3. удалить значение (`.delete()`)

uniqueNumbers.delete(10);

// 4. очистить (`.clear()`)

uniqueNumbers.clear();

// 5. узнать размер (`.size`)

uniqueNumbers.size;
```

### перебор

```js
for (const valueItem of uniqueValues) {
  console.log(valueItem);
}
```

### главный кейс - удаление дубликатов из массива

```js
const numbersArray = [1, 2, 2, 3, 3, 4];

const uniqueResult = [...new Set(numbersArray)];

console.log(uniqueResult); // [1, 2, 3, 4]
```

### нюанс

```js
const testSet = new Set();

testSet.add({ name: 'Alex' });
testSet.add({ name: 'Alex' });

console.log(testSet.size); // 2 - потому что объекты сравниваются по ссылке, а не по содержимому
```

### когда использовать

- нужно хранить уникальные значения
- нужно быстро проверить наличие

## 2. `Map`

- это коллекция пар ключ-значение, но в отличие от объекта:
  - ключ может быть любого типа
  - сохраняется порядок
  - есть удобные методы

```js
// 1. создание

const userMap = new Map();

// 2. добавление

userMap.set('name', 'Alex');
userMap.set(1, 'number key');

// 3. получение

userMap.get('name'); // 'Alex'

// 4. проверка

userMap.has('name'); // true

// 5. удаление

userMap.delete('name');

// 6. размер
userMap.size;
```

### перебор

```js
for (const [keyItem, valueItem] of userMap) {
  console.log(keyItem, valueItem);
}
```

### главное отличие от объекта

```js
const userObject = {};

userObject[{}] = 'value';

console.log(userObject); // { "[object Object]": "value" }
// объект превращается в строку ↑

const mapExample = new Map();

const objKey = {};
mapExample.set(objKey, 'value');

console.log(mapExample.get(objKey)); // "value"
// ключ остаётся объектом
```

### когда использовать:

- ключи не строки
- важен порядок
- часто добавляются/удаляются данные
- нужен быстрый доступ
