# difference between assign and spread operator

## 1. что сравниваем

- Object.assign() и spread оператор оба используются для:
  - копирования объектов
  - объединения объектов

## 2. `Object.assign()`

- это метод объекта Object, который копирует свойства из одного объекта в другой

### синтаксис

```js
Object.assign(target, source1, source2, ...)
```

### пример

```js
const userData = { name: 'Alex' };
const userExtra = { age: 25 };

const mergedUser = Object.assign({}, userData, userExtra);

console.log(mergedUser); // { name: 'Alex', age: 25 }
```

### важно

1. первый аргумент (target) **изменяется**
2. возвращается тот же самый объект

## 3. Spread оператор

- это более современный синтаксис (ES6+)

```js
const baseSettings = { theme: 'dark' };
const extraSettings = { fontSize: 16 };

const fullSettings = {
  ...baseSettings,
  ...extraSettings,
};

console.log(fullSettings); //{ theme: 'dark', fontSize: 16 }
```

### что происходит

`...baseSettings` разворачивает объект в пары: `theme: 'dark'`

## 4. главная разница

1. мутация

- `Object.assign` изменяет объект

```js
const objA = { x: 1 };

Object.assign(objA, { y: 2 });
```

- spread создаёт новый объект

```js
const objB = { x: 1 };

const newObj = { ...objB, y: 2 };
```

2. синтаксис

- `Object.assign()` - функция
- `...` - синтаксис языка

3. читаемость

```js
// хуже читается
Object.assign({}, obj1, obj2);

// лучше читается
{ ...obj1, ...obj2 }
```

## 5. поверхностное копирование

- оба делают shallow copy, а не глубокую

```js
const originalProfile = {
  name: 'Alex',
  address: {
    city: 'Amsterdam',
  },
};

const copiedProfile = { ...originalProfile };

copiedProfile.address.city = 'Paris';

console.log(originalProfile.address.city); // Paris
```

- потому что копируется ссылка на тот же объект, а не сам вложенный объект

## 6. перезапись свойств

- если ключи совпадают, побеждает последний

```js
// spread
const resultObject = {
  ...{ a: 1 },
  ...{ a: 2 },
};

console.log(resultObject); // { a: 2 }

// Object.assign
const resultAssign = Object.assign({}, { a: 1 }, { a: 2 });

console.log(resultAssign); // { a: 2 }
```
