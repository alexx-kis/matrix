# nullish coalescing operator `??`

- это оператор, который позволяет задать значение по умолчанию, но только если слева `null` или `undefined`

```js
value1 ?? value2;
```

- если `value1` равен `null` или `undefined`, берётся `value2`, иначе `value1`

## пример

```js
const userName = null;
const resultName = userName ?? 'Guest';

console.log(resultName); // "Guest"
```

- `userName === null` → значит берётся правая часть "Guest"

```js
const ageValue = 25;
const resultAge = ageValue ?? 18;

console.log(resultAge); // 25
```

- здесь `ageValue` не `null` и не `undefined`, значит возвращается оно само

## главное отличие от `||`

- оператор `||` считает falsy:
  - 0
  - ''
  - false
  - NaN
  - null
  - undefined

- оператор `??` вернёт левое значение, только если правое null или undefined

## важный нюанс

- нельзя смешивать ?? с || или && без скобок вот так

```js
const testValue = null ?? 10 || 20;
```

- нужно так

```js
const testValue = (null ?? 10) || 20;
```
