# flat, flatMap, includes, Array.from() methods

## 1. flat()

- "разглаживает" вложенные массивы (убирает вложенность)

```js
const numbersNested = [1, 2, [3, 4]];

const numbersFlattened = numbersNested.flat();

console.log(numbersFlattened); // [1, 2, 3, 4]
```

- по умолчанию `flat()` убирает один уровень вложенности
- можно указать уровень

```js
const numbersDeepFull = numbersDeep.flat(2);

console.log(numbersDeepFull); // [1, 2, 3, 4]
```

- полностью "расплющить"

```js
const numbersInfinity = numbersDeep.flat(Infinity);
```

### важно знать

- `flat()` не изменяет исходный массив, а возвращает новый массив

## 2. flatMap()

- map + flat(1)

- то есть сначала делает `map`, потом сразу "разглаживает" результат

```js
const wordsArray = ['hello', 'world'];

const letters = wordsArray.flatMap((wordItem) => {
  return wordItem.split('');
});

console.log(letters); // ['h', 'e', 'l', 'l', 'o', 'w', 'o', 'r', 'l', 'd']
```

- то же самое через map + flat:

```js
const lettersStep = wordsArray
  .map((wordItem) => {
    return wordItem.split('');
  })
  .flat();
```

### когда используют

- чтобы развернуть массив массивов после трансформации

### важно

- `flatMap()` делает `flat` только на 1 уровень, глубину задать нельзя

## 3. `includes()`

- проверяет, есть ли элемент в массиве

```js
const fruitsArray = ['apple', 'banana', 'orange'];

console.log(fruitsArray.includes('banana')); // true
console.log(fruitsArray.includes('grape')); // false
```

- вторым аргументом можно указать, с какого индекса искать

```js
const numbersList = [1, 2, 3, 4];

console.log(numbersList.includes(2, 2)); // false
```

## 4. `Array.from()`

- создаёт массив из псевдомассива или iterable

- псевдомассивы - это объекты вроде `arguments`, `NodeList`, `HTMLCollection`, строки

```js
const lettersFromString = Array.from('hello');

console.log(lettersFromString); // ['h', 'e', 'l', 'l', 'o']

const arrayLikeObject = {
  0: 'a',
  1: 'b',
  length: 2,
};

const realArray = Array.from(arrayLikeObject);

console.log(realArray); // ['a', 'b']
```

### с `map` внутри

- `Array.from()` умеет сразу трансформировать:

```js
const numbersGenerated = Array.from([1, 2, 3], (numItem) => {
  return numItem * 2;
});

console.log(numbersGenerated); // [2, 4, 6]
```

### генерация массива

```js
const sequenceArray = Array.from({ length: 5 }, (_, indexValue) => {
  return indexValue;
});

console.log(sequenceArray); // [0, 1, 2, 3, 4]
```
