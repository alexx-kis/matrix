# keys, values, fromEntries, entries methods

## 1. объект в JS

- это структура данных вида:

```js
const userProfile = {
  name: 'Alex',
  age: 25,
  isAdmin: true,
};
```

- здесь:
  - `"name"`, `"age"`, `"isAdmin"` - ключи (keys)
  - `"Alex"`, `25`, `true` - значения (values)

## 2. Object.keys()

- возвращает массив всех ключей объекта

```js
const productData = {
  title: 'Laptop',
  price: 1200,
  inStock: true,
};

const productKeys = Object.keys(productData);

console.log(productKeys);
```

- вывод:

```
["title", "price", "inStock"]
```

### когда используется

- перебрать объект
- посчитать количество свойств
- сделать фильтрацию

## 3. Object.values()

- возвращает массив всех значений

```js
const orderInfo = {
  id: 101,
  total: 250,
  status: 'paid',
};

const orderValues = Object.values(orderInfo);

console.log(orderValues);
```

- вывод:

```
[101, 250, "paid"]
```

### когда используется

- сложить числа
- проверить значения

## 4. Object.entries()

- возвращает массив пар `[ключ, значение]`

```js
const carInfo = {
  brand: 'BMW',
  year: 2022,
};

const carEntries = Object.entries(carInfo);

console.log(carEntries);
```

- вывод:

```
[
  ["brand", "BMW"],
  ["year", 2022]
]
```

- можно работать, как с массивом
- - использовать деструктуризацию

```js
const userSettings = {
  theme: 'dark',
  fontSize: 16,
};

const userEntries = Object.entries(userSettings);

userEntries.forEach(([settingKey, settingValue]) => {
  console.log(settingKey, settingValue);
});
```

## 5. Object.fromEntries()

- делает обратное - из массива пар -> объект

```js
const entriesList = [
  ['name', 'Alex'],
  ['age', 25],
];

const newUserObject = Object.fromEntries(entriesList);

console.log(newUserObject);
```

- вывод:

```js
{
  name: "Alex",
  age: 25
}
```

## кейс

- удалить falsy значения

```js
const rawFormData = {
  name: 'Alex',
  email: '',
  age: null,
  isActive: true,
};

const cleanData = Object.fromEntries(
  Object.entries(rawFormData).filter(([fieldKey, fieldValue]) => {
    return Boolean(fieldValue);
  }),
);

console.log(cleanData);
```
