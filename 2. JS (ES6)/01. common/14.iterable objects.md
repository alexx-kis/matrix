# iterable objects

- в JS есть объекты, по которым можно проходиться поэементно:
  - массивы
  - строки
  - Map
  - Set

## главная идея

- iterable object - это объект, который умеет по запросу отдавать значения одно за другим
- JS это делает через специальный механизм - итераторы

- обычный объект `{}` не умеет сообщать JS, как по нему проходить, в отличие от массива, который знает:
  - с чего начать
  - как получить следующий элемент
  - когда закончить

## что отвечает за это?

- специальный метод `Symbol.iterator`

## что такое `Symbol.iterator`

- это специальный встроенные Symbol, который говорит JS: "вот функция, которая создаёт итератор"

## важнейшая схема

- iterable object выглядит вот так:

```js
const someObject = {
  [Symbol.iterator]() {
    return iteratorObject;
  },
};
```

- а iteratorObject выглядит так:

```js
const iteratorObject = {
  next() {
    return {
      value: ...,
      done: ...,
    }
  }
}
```

## что делает next()

- каждый вызов `next()` возвращает объект:

```js
{
  value: текущее_значение,
  done: false
}
```

- когда значения заканчиваются:

```js
{
  done: true;
}
```

## посмотрим вручную

### итератор массива

```js
const colorsArray = ['red', 'green', 'blue'];

const colorsIterator = colorsArray[Symbol.iterator]();

console.log(colorsIterator.next());
console.log(colorsIterator.next());
console.log(colorsIterator.next());
console.log(colorsIterator.next());
```

- вывод:

```js
{ value: 'red', done: false }
{ value: 'green', done: false }
{ value: 'blue', done: false }
{ value: undefined, done: true }
```

### что произошло

`colorsArray[Symbol.iterator]()` создал объект-итератор, а `next()` начал выдавать значения по одному

### как работает `for...of` внутри

```js
for (const colorValue of colorsArray) {
  console.log(colorValue);
}
```

- JS примерно делает так:

```js
const internalIterator = colorsArray[Symbol.iterator]();

while (true) {
  const iteratorResult = internalIterator.next();

  if (iteratorResult.done) {
    break;
  }

  console.log(iteratorResult.value);
}
```
