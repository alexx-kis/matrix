# Шаблонные строки

- это специальный способ создавать строки в JS, который:
  - удобнее обычных кавычек
  - позволяет вставлять переменные прямо внутрь строки
  - поддерживает переносы строк без `\n`

## 2. обычные строки (как было раньше)

- раньше строки создавались так:

```js
const userName = 'Alex';
const greetingText = 'Hello, ' + userName + '!';
```

- здесь используется конкатенация (`+`) - склеивание строк
- это работает, но:
  - не удобно
  - трудно читать при большом тексте

## 3. template string - новый способ

```js
const userNameValue = 'Alex';
const greetingMessage = `Hello, ${userNameValue}!`;
```

## 4. вставка переменных

1. переменные:

```js
const ageValue = 25;

const textResult = `Age: ${ageValue}`;
```

2. выражения:

```js
const numberA = 10;
const numberB = 5;

const sumText = `Sum: ${numberA + numberB}`;
```

3. вызов функций:

```js
function getUserName() {
  return 'Alex';
}

const resultText = `User: ${getUserName()}`;
```

## 5. многострочные строки

```js
// без template string:
const textOld = 'Line 1\nLine 2\nLine 3';

// с template string:
const textNew = `
Line 1
Line 2
Line 3
`;
```
